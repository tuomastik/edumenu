# Edumenu
Edumenu is a Windows Phone application showing restaurant menus for university restaurants in Tampere, Finland. The app supports Windows Phone 8, 8.1 and Windows 10 Mobile.

Edumenu in Windows Store: https://www.microsoft.com/fi-fi/store/p/edumenu/9wzdncrdndsg

![](graphics/edumenu_demonstration.gif)

## Kuvaus

Edumenu tarjoaa Tampereen oppilaitoksien TTY, TAY, TAMK ja TAKK viikon ruokalistat.

Oppilaitokset ja ravintolat:

 - **TTY**: Reaktori, Hertsi, Newton
 - **TAY**: Yliopiston Ravintola, Minerva, Linna, CafÈ & Lunch Pinni, CafÈ Alakuppila
 - **TAMK**: Pirteria, Ziberia, Live, Live - Fusion Kitchen
 - **TAKK**: Nasta, Nasta - Fusion Kitchen, Ratamo
